## What is This?

This guide helps you use **Claude Code** completely **free** using **Bonsai CLI** — without any API key or paid subscription.

Bonsai provides access to frontier AI models (including Claude) in **stealth mode**, where model names are hidden to ensure unbiased evaluation.

| Tool | Purpose |
|---|---|
| **Claude Code** | The AI coding assistant (interface) |
| **Bonsai CLI** | Provides free access to frontier AI models |

> **No API key required. No credit card needed.**

## Requirements

| Requirement | Version |
|---|---|
| Node.js | v18 or higher |
| npm | Comes with Node.js |
| Terminal | Windows / Linux / macOS |

**Check if already installed:**

```bash
node -v
npm -v
```

✅ You should see version numbers for both.

> **Don't have Node.js?** Download and install the LTS version from [https://nodejs.org](https://nodejs.org)

---

## Step 1: Install Bonsai CLI

Install Bonsai CLI globally using npm:

```bash
npm install -g @bonsai-ai/cli
```

**Verify installation:**

```bash
bonsai --version
```

✅ You should see a version number printed.

---

## Step 2: Login to Bonsai

Authenticate your terminal with your Bonsai account:

```bash
bonsai login
```

**What happens:**
- A browser window opens automatically
- Log in or sign up for a free Bonsai account
- Your terminal gets authenticated

✅ Once done, you are ready to use Claude Code for free!

---

## Step 3: Start Claude Code

Start Claude Code using Bonsai:

```bash
bonsai start claude
```

**What this command does:**
- Launches a Claude Code session
- Automatically assigns a frontier AI model
- Runs in **stealth mode** (model name is hidden for unbiased evaluation)
- No API key required

🎉 That's it! You are now using Claude Code for free!

---

## Step 4: Logout (Optional)

When you are done, logout from Bonsai anytime:

```bash
bonsai logout
```
